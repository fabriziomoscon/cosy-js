Getting Started With Snuggle
============================

## Bootstrapping ##

```js
snuggle = require('cosy-js').snuggle;

controls = require('/path/to/your/controls');
libs = require('/path/to/your/libraries');

debug = false;

snuggle.up($('html'), controls, libs, debug);
```
